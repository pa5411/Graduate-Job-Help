Cover Letter

The cover letter is a short document which complements a CV and demonstrates the reasons an individual is interested in applying to both the role and the company. There are a wide range of opinions as to how best to write a cover letter. This guide outlines one typical approach.

Layout

General Layout

In a formal letter the sender's address is right justified whilst the recipient's address is left justified. The date at the time the letter is written is also usually added. The recipient should be addressed by name if known, otherwise an appropriate salutation may include (amongst others) "To whom it may concern" or "Dear Sir or Madam". Following the salutation there should be one line which states clearly the intent of the letter, which in this context is an application for a specific role.

There are then usually three key sections of a cover letter. The first section covers the background history of the individual. This includes their academic degree(s) as well as any recent work experience and is therefore only a small number of sentences long. The next section describes the reasons for which an individual is making an application to both the role and the company. This will usually be a sizeable paragraph.

The final section describes in more detail how the skills and experiences of the individual meet the requirements of the role and match the values of the company. This will usually be the largest section of the cover letter and may be two or three paragraphs.

The cover letter should end with a closing summary, drawing the letter to a close. If the recipient is known by name, "Yours sincerely" should be used, otherwise use "Yours faithfully". In most cases the cover letter should not exceed a single side of A4.

The layout of a typical cover letter is shown below for a fictious individual named John Doe applying to the fictitious Company XY and addressed to the fictitious recipient Jane Doe:

John Doe
London

January 01, 2020

Company XY
London

Dear Jane Doe

This is an application for the XYZ Graduate Programme (Ref #1234) as advertised on Company XY's career website.

...

Finally, I am flexible to other opportunities that you may have available. I have attached a CV for your consideration to support my application. I thank you for your time and look forward to hearing from you.

Yours sincerely,

John Doe

Format

A cover letter should be written in a document writing software such as MS Word or LaTeX. Since assessors may view the document using different software and platforms, it is desirable for the document to be submitted using the .pdf format. If this format is used, make sure that the text is not embedded so that a scanning tool can read the text easily. In practical terms this means that when the document is opened in this format, the text can be highlighted and copied easily. Always check the final document before submitting. This ensures that the formatting has not changed. If changes are needed, make them using the original software that was used to first create the document. Generally the following rules should be followed:

  • Use typical fonts such as Times New Roman, Liberation Serif or Computer Modern.
  • Use black colour only.
  • Use bold or italics sparingly.
  • Avoid using tables for formatting as this may interfere with the scanning tool.
  • Avoid using any graphics.
  • Avoid very narrow page margins.
  • Avoid using boarders.
  • Use paragraphs instead of bullet points.
  • Include your surname when choosing the document file name. For the person named John Doe, one example is "JDoeCoverLetter.pdf".

Language style

  • Use plain simple English, i.e. avoid using obscure words and phrases unless necessary.
  • Avoid text speak and American English.
  • Abbreviations not universally known should be explained when first used in the cover letter.
  • Use numbers to quantify the scope of an achievement, e.g. "presented to 30 people".
  • Use the first person narrative.
